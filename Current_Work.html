Molecular Playground Expansion

Recently I completed a class project called Molecular Playground Expansion which can be found here. You can visit the site www.teamdropout.com to see the prototype of this project. This was project for my software engineering class that was aimed to teach us about how to create software design documents for a large code base. At Umass there is a projector that displays 3D molecules that anyone can walk up to and rotate using a kinetic. This was called Molecular Playground and there are installations like the one at Umass across the globe. I worked with a team of ten to design and implement Molecular Playground Expansion which would be a web version of the old system that would centralize and update all installations through this one web app. I personally worked on the database code for this website using postgresql and nodejs to store all of the data in our system.
