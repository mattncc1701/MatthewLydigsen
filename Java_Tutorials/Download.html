How to use Java

Now that you have java the question becomes how do you actually write code and use java. Well you can use command line which I will have a section on in the advanced topics section. But I imagine many of the people reading this aren't familiar with command line so we will download an Integrated Development Enviorment (IDE). Two popular java IDE's are Netbeans and Eclipse, you can also use Dr. Java but I personally like Eclipse and it has a lot of nice features which I will go through in later sections.
