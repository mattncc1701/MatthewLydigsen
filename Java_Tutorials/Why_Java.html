Answer

Well there is no exact answer. Every language you will learn is different and has strengths and weaknesses. Javascript is great for programming websites, C++ is used for a lot of graphics based coding. So what are java's strengths. Well Java, unlike most languages, catches a lot of errors before you run your code so you can get use to java's syntax more quickly and catch errors more easily. Java is also relatively easy to learn and uses Object Oriented Programming (which will be discussed in greater detail in a later section). Java is also transferable between almost any computer so whether you are working on a pc or a mac you will be able to run the same program. These reasons make java a great language to learn first.
