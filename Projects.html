Downtime Capture System

This was a Downtime Capture System, using a Raspberry pi, I created to measure when a factory machine stopped functioning. It would index the time that the machine went down by receiving a signal from the machine. Then a worker could use a GUI i created to record the reasons why the the machine went down. The reasons it could go down could be updated at any time from an excel spread sheet. Then once the event had been logged it would send the data to a different excel spread sheet that could be downloaded and processed at any time.

Space Invaders Game

Using java, I recreated the old arcade game space invaders. In past summers I taught a free java programming course using this program. Each week I would introduce new concepts about java and let high school kids who were taking this class fill in the code to this game. By the end of the class the students would have a firm knowledge of java and will have programmed much of the game Space Invaders. The image shown is a snap shot of my game. I'm in the process of converting my code into a java applet so you can play it on my website.
